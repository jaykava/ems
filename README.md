# Project Setup Guide

## Requirements
- **Ruby**: `3.4.5`  
- **Rails**: `8.0.2`  
- **Database**: PostgreSQL (or your preferred adapter)  

## Setup Instructions

Install dependencies

```bundle install```

Set up the database

```rails db:create```
```rails db:migrate```
```rails db:seed```

Running the Server

Start the Rails development server with:

```rails server```


Visit http://localhost:3000
 in your browser.
